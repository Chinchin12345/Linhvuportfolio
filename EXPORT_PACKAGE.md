## 🎯 Quick Start Guide

### Để Xem Portfolio:
```bash
# Option 1: Mở trực tiếp
Double-click vào index.html

# Option 2: Local server (khuyên dùng)
cd "Cursor Port"
python3 -m http.server 8000
# Mở: http://localhost:8000
```

### Để Đưa vào Figma:

**🚀 Nhanh nhất (5 phút):**
1. Mở `index.html` trong browser
2. Chụp full-page screenshot (dùng extension)
3. Import screenshot vào Figma làm reference
4. Trace theo DESIGN_GUIDE.md

**⚡ Tự động (15 phút):**
1. Start local server: `python3 -m http.server 8000`
2. Mở Figma → Install plugin "html.to.design"
3. Plugin → Nhập: `http://localhost:8000`
4. Import và chỉnh sửa

**✨ Professional (1-2 giờ):**
1. Đọc `DESIGN_GUIDE.md`
2. Setup Design System trong Figma
3. Import images từ `assets/images/`
4. Recreate từng section với Auto Layout
